# Multi-Threaded Port Scanner with Banner Grabbing

This project is a Python-based network port scanner designed for defensive security purposes.

## Features
- Multi-threaded TCP port scanning
- Service name detection
- Banner grabbing for open ports
- Ethical usage warning
- Scan result export to file

## Usage
python defence_port_scanner.py

## Disclaimer
This tool is strictly for educational purposes.
Unauthorized scanning of networks is illegal.
